Welcome to Your application accessing S4 within your private Azure VNet!

This is your SAP Cloud Platform Cloud Foundry TomEE application powered by the SAP Cloud SDK.

  • Change to the application/ directory and execute mvn package tomee:run.
  • Visit the BTPAzureProxyServlet. You can find its source code in application/src/main/java/.
  • Additional static content can be placed inside application/src/main/webapp/.

Further help and examples are available in the SAP Cloud SDK documentation.
