Welcome to Your application accessing S4 within your private Azure VNet!

This is your SAP Cloud Platform Cloud Foundry TomEE application powered by the SAP Cloud SDK to access Azure via the private link service.

Find the project docs on our GitHub repos.

  • Change to the application/ directory and execute mvn package tomee:run. Afterwards open localhost:8080 (in SAP BAS only from the shell. Locally you can navigate in your browser)
  • Visit JWT Debug
  • Visit SAP Ping for initial testing.
  • Visit BTPAzureProxyServlet for Cloud SDK.
  • Visit BTPAzureProxyServletIgnoreSSL for plain Apache Http with SSL ignore.
  • Visit BTPAzureProxyServletPrincipalPropagation for manual SAP Principal Propagation.
  • Visit BTPAzureProxyServletRFC for RFC connections.
  • You can find its source code in application/src/main/java/.
  • Additional static content can be placed inside application/src/main/webapp/.

Further help and examples are available in the SAP Cloud SDK documentation.
